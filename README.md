### What is this repository for? ###

* ACME Secure Browser is a demo app which uses the 1Password for iOS 8 Extension.

### How do I get set up? ###

* Make sure you use 1Password 4.6b4 or later
* Make sure you use Xcode6-Beta-3 or later
